# BAZA SVL — Лендинг деревообрабатывающей базы

## О проекте

Одностраничный лендинг для производственной базы **BAZA SVL** (деревообработка полного цикла).  
Стек: чистый HTML + CSS + Vanilla JS, без фреймворков. Открывается прямо из файла в браузере.

**Сайт:** https://profikate.github.io/baza-svl/  
**Репозиторий:** https://github.com/ProfiKate/baza-svl

## Дизайн-система

### Цвета (CSS-переменные в `style.css`)

| Переменная       | Значение    | Назначение                          |
|------------------|-------------|-------------------------------------|
| `--graphite`     | `#1e1e1e`   | Основной фон                        |
| `--graphite-2`   | `#2a2a2a`   | Фон тёмных секций, карточек         |
| `--graphite-3`   | `#333333`   | Карточки услуг, форма               |
| `--oak`          | `#c8a96e`   | Акцент «натуральный дуб»            |
| `--oak-bg`       | `#f5ede0`   | Фон секции «Сотрудничество»         |
| `--amber`        | `#d4820a`   | CTA-кнопки, иконки, метки          |
| `--amber-hover`  | `#b86d00`   | Hover-состояние amber               |
| `--white`        | `#ffffff`   | Текст на тёмном, карточки продукции |
| `--text-dark`    | `#2c2318`   | Текст на светлом фоне               |
| `--text-muted`   | `#8a7a6a`   | Вторичный текст                     |

### Шрифты

- **Montserrat** — заголовки, навигация, кнопки (font-weight 700–800)
- **Manrope** — основной текст, подписи (font-weight 400–600)
- Подключены через Google Fonts

## Секции лендинга

### 1. Header (фиксированный)
- Логотип слева + тег «Деревообработка полного цикла»
- Навигация: О нас | Услуги | Продукция | Сотрудничество | Контакты
- При скролле > 40px становится тёмным с blur-эффектом
- На мобайле — бургер-меню
- Активный пункт меню подсвечивается при скролле по секциям

### 2. Hero
- Полноэкранный (100svh), фон — `Фото основное.jpg`
- Анимация zoom при загрузке (CSS keyframes `heroZoom`)
- Тёмный градиентный оверлей
- Заголовок «BAZA SVL», подзаголовок, две кнопки CTA
- Анимированная стрелка «листать вниз»

### 3. О нас (`#about`)
- Сетка 3 колонки × 2 ряда — тёмные карточки (`--graphite-2`) с иконкой (amber) сверху
- Hover: карточка приподнимается, янтарная линия выезжает сверху слева направо
- Порядок карточек: Работаем с 2012 / Полный цикл / Строго по требованиям → Прогнозируемый результат / Надёжное партнёрство / Принципиальное качество
- Фон: `--graphite` (основной)

### 4. Услуги (`#services`)
- Сетка 3 карточки, фон `--graphite-2`
- Каждая карточка: видео в круглой рамке 200px (двойное янтарное свечение) + детали
- Лейблы «Оборудование / Для кого / Преимущества» — белый текст uppercase с дубовым кружком-маркером слева
- Три услуги: Распил / Вакуумная сушка / Калибровка и профилирование
- Видео воспроизводятся/паузятся через IntersectionObserver (экономия ресурсов)

### 5. Продукция (`#products`)
- Заголовок: «Готовые изделия»
- Сетка 3 колонки, фото-карточки с hover zoom
- 6 категорий: Пиломатериалы, Погонаж, Столярные изделия, Эксклюзив, Заготовки, Технологические поддоны
- Фон: `--graphite` (основной)

### 6. Сотрудничество (`#cooperation`)
- Сетка 2×2, фон `--oak-bg` (светлая секция)
- 4 варианта работы: Подряд / Индивидуальное производство / Аренда мощностей / Спецпредложение
- Карточка «Спецпредложение» выделена золотой рамкой и бейджем

### 7. Контакты (`#contacts`)
- Три равные колонки в одном ряду, одинаковой высоты (`align-items: stretch`)
- Колонка 1: контакты — телефон + 3 кнопки мессенджеров (Viber/Telegram/WhatsApp), email, Instagram, адрес
- Колонка 2: форма заявки (`contact-form`)
- Колонка 3: Google Maps с реальными координатами базы `53.4205527, 27.0466418`, спутниковый вид (`t=k`) + кнопка «Проложить маршрут»
- На ≤1024px — одна колонка (стопкой)
- Фон: `--graphite-2`

### 8. Footer
- Логотип + копирайт + иконки соцсетей (Instagram, Viber, Telegram)

## Логика формы заявки (`js/script.js`)

Кнопка `type="button"` с `onclick="handleFormSubmit()"` — глобальная функция в `script.js`.  
При заполненных имени и телефоне: форма заменяется success-блоком с сообщением «Заявка принята!» прямо на месте (без скролла).  
Внутри success-блока — кнопка «Написать в Telegram» с предзаполненным текстом заявки (имя + телефон + сообщение).
